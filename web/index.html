How it works

  1. Dispatcher adds a load and assigns a driver.
  2. Driver gets a text with a secure link to a status page.
  3. Driver taps status (En‑route, Arrived, Loaded...). We collect ETA if appropriate.
  4. We notify agents/merchants/receivers based on the status—no spam to the wrong folks.
